About the Project

USAVE2 ("Understanding Saving in Europe 2") is an Erasmus+ strategic partnership project focused on personal finance literacy. It builds on the success of UNPIE2 (Understanding Pensions in Europe 2) and extends it to the full life-cycle savings decision.

The project delivers a suite of interactive web applications that allow students and citizens to explore key concepts in personal finance: human capital, optimal consumption smoothing, disability insurance, portfolio construction, and stochastic retirement planning.

Erasmus+ Project ID: 2019-1-LI01-KA203-000125
